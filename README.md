# Warehouse Management System

This project is a warehouse management simulator designed to model real world inventory processes. It tracks stock movement from suppliers through the warehouse to customers and provides information about inventory state.

## Overview

Flow:

**Supplier → Warehouse → Customer**

It allows users to:

* View current stock levels
* Track outgoing items
* Monitor remaining inventory

## Features

### Tier 1 (Core Functionality)

* Bulk inventory tracking
* Quantity-based stock management

### Tier 2

* FIFO (First-In, First-Out) batch handling
* Ensures older stock is dispatched first

### Tier 3

* Individual item tracking
* Control and traceability

## Functionality

* Add items to inventory
* Remove items according to tier logic
* Display stock status, item counts, and relevant details
